# airport-checkin-optimization
An interaction Streamlit dashboard that optimizes airport check-in desk allocation using static vs dynamic resource modelsto reduce passenger overload during peak hours 
## Features
- Generate synthetic airport data
- Explore data with EDA (graphs, stats)
- Optimize check-in desk allocation
- Interactive Streamlit dashboard

## Usage
- Use sliders or input fields to change number of check-in desks or passenger flow.
- Dashboard automatically updates wait times and visualizations.

- `data_generator.py` – Generates synthetic airport data  
- `synthetic_airport_data.csv` – Sample dataset  
- `eda.py` – Exploratory data analysis  
- `optimizer.py` – Optimization logic for check-in desks  
- `app.py` – Streamlit dashboard
